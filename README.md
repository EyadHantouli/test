## How migration done

Created python sctipts located at "src/lib/generators"
Make sure to run them every time one of this files is modified
    ```1-    التّفْصيْلُ الموضوعِي.txt```
    ```2-  any file in src/templets folder.```

_If any error occurs during running try to run them normally by double-clicking on the file._
